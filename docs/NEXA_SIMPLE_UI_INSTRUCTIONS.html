Vendor adminThe private dashboard for one vendor.

Super Admin

Open admin

  1. Open your web browser.
  2. Go to your NEXA site and add /admin to the address.
    Example: https://www.nexa-mm.com/admin
  3. Sign in with the email and password your owner provided.

Screen layout

  • Left: main menu (sections).
  • Top: search, notifications, profile.
  • Center: work area — changes when you select a menu item.

If you do not see Finances or Settings, your account role is limited by design.

Vendors

  1. Click Vendor in the left menu.
  2. Search or filter the list as needed.
  3. Use Review applications for new signups — there is no manual Add vendor button.
  4. Use the row menu (⋮) for profile or storefront preview when available.

Settings and team

  1. Click Settings.
  2. General — platform name and logo.
  3. Users — add or edit staff (owner only).
  4. Activities — audit timeline of admin actions.

Vendors

Sign in

  1. Open https://www.nexa-mm.com/vendor/login.
  2. Enter your vendor email and password.

Admin URL

Bookmark your vendor admin, for example:

  • https://yourstore.nexa-mm.com/admin
  • https://www.nexa-mm.com/vendor/your-store-slug/admin

Preview and share

  1. In vendor admin, use Preview or View storefront.
  2. Share your vendor URL with customers — not the main site product page.
